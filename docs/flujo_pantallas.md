Registro de pantallas para lograr con el objetivo del mvp:
- FrmLogin: Control de acceso. Campos para usuario (DNI o nombre de usuario), contraseña y botón de ingreso.

- FrmPrincipal: Contenedor principal (puede ser un JFrame con un menú superior o botones laterales). Desde aquí se accede a todos los módulos según el rol del usuario logueado.

- FrmGestionEstudiantes: Módulo para registrar, editar y dar de baja a estudiantes, así como registrar y vincular a sus apoderados (pestañas o paneles separados).

- FrmGestionColaboradores: Mantenimiento del personal (docentes, secretarias, directores) y asignación de sus roles en el sistema.

- FrmConfiguracionAcademica: Pantalla exclusiva para Dirección. Sirve para aperturar el año lectivo, registrar grados, secciones y definir la cantidad de vacantes.

- FrmAsignacionDocente: Interfaz para vincular a un docente registrado con un grado, sección y asignatura específica.

- FrmMatricula: El núcleo transaccional del MVP. Interfaz para buscar un estudiante y matricularlo en un año y grado disponible.
